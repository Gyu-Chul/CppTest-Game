# C-Test-Game
Simple Game by C++


2025-01-19 (일요일)

현재 진척도 :
1. 게임 진행을 위한 클래스 다이어그램 구조화 성공
2. 빈 상태로(테스트 용도로) 정의된 클래스 인스턴스 생성완료
3. 소프트웨어 프로세스 및 개발 다이어그램 작성및 논의 완료


다음주 신민철의 업무
* C++ 게임의 인프라 구축 및 그래픽 모듈 코드 생성
* 타이틀 세션, 일반 세션, DB 코드 , GP코드 역할 담당
* 인프라 및 그래픽 시스템 주 관리자


다음주 최규호의 업무
* C++ 인게임 플레이 구체화
* 상점 세션, 스테이지 세션, 시스템 세션 역할 담당
* 클래스 다이어그램 , 일반 시스템 주 관리자
* 일반 시스템 코드 작성시, 타이틀과 일반 세션 연결 로직은 보류


다음주 목표 : C++ 게임 완성하기

신민철의 한마디 : 
3시간은 어떻게든 내가 알아서 채워올게
타이틀 세션,일반 세션 일단 선언 부분 과 간단한 정의 그리고 데이터 세션 먼저 완성 하기.
